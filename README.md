> GitHub を使用した、日々の学習&アウトプットの習慣化サービス 🫐⚔️

簡単なデイリークエストで、コツコツ学習&発信を習慣化する。<br/>
そんな学習スタイルを身につけるための、クエストアプリです！

<br>


## 概要

Blueberry Quest は、<br>
多くの開発者が実践する取り組みを、まとめました：

- 日常的に Git でのバージョン管理を活用しよう
- GitHub に草を早さそう
- 記事を執筆しよう

、、などなど、<br>
効果的な学習と、アウトプットなどを、毎日コツコツ習慣化できます🙌

<br>

## 特徴

- 🎥 レクチャー動画で気軽に学習
- ⚔️ 習慣化を重視した、デイリークエスト
- 💚 仲間とモチベーションを高め、つながりを作る

### これは誰のためのものですか？

- 学習や個人開発の継続に課題を感じている方
- アウトプット・発信に、苦手意識を持っている方
- モチベーションを高めて、作業に取り組みたい方
- もしくは、同じようなの開発者/学習者とのコラボレーションを楽しみたい方

### このプロジェクトの目的は？

- 毎日コツコツ学習・アウトプットを習慣化することで、スキル習得やキャリア的な目標達成を支援すること
- 独学では、おろそかにしがちな慣習を早い段階で取り入れるきっかけを作ること
- 初心者同士が、交流する手助けをすること

<br>
